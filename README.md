このプロジェクトは、アイドルのライブ情報をまとめたカレンダーを作成することを目的としています。

## インストール

vscode + devcontainer を用いて開発を行うことを想定しています。

## 使い方

以下を用いて開発サーバーを起動することができます。

```bash
yarn dev
```

## 環境

- node v20.10.0
- yarn v1.22.19
- next v14.0.4
- react v18
- typescript v5
- eslint v8.0.1
